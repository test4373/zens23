📋 ADIMLAR:

1. https://render.com → Sign up (GitHub ile)

2. Dashboard → New → Web Service

3. GitHub repo seç veya Create new repo

4. Ayarlar:
   Name: zenshin-backend
   Region: Frankfurt (EU Central)
   Branch: main
   Root Directory: BACKEND
   Runtime: Node
   Build Command: npm install
   Start Command: node server.js
   Plan: FREE

5. Environment Variables ekle:
   NODE_ENV = production
   PORT = 10000 (otomatik)
   
   (Opsiyonel - Google Drive için):
   GDRIVE_CLIENT_ID = your-client-id
   GDRIVE_CLIENT_SECRET = your-secret
   GDRIVE_REFRESH_TOKEN = your-token

6. Create Web Service → DEPLOY! ✅
